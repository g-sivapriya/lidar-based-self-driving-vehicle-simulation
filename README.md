# LiDAR-Based Self-Driving Vehicle Simulation

## Overview
This project is a web-based simulation of a LiDAR-enabled autonomous vehicle developed using HTML5 Canvas and JavaScript. The simulation demonstrates obstacle detection, traffic awareness, collision avoidance, and autonomous lane-following behavior.

## Features
- Simulated 360° LiDAR scanning
- Obstacle detection and avoidance
- Vehicle-to-vehicle collision prevention
- Dynamic traffic generation
- Autonomous lane following
- Real-time visualization using HTML5 Canvas

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Canvas API

## How It Works
1. Vehicles are generated dynamically in different lanes.
2. Each vehicle uses simulated LiDAR rays to scan its surroundings.
3. Obstacles and nearby vehicles are detected.
4. Vehicles automatically adjust speed and direction to avoid collisions.
5. The system maintains lane discipline while navigating traffic.
